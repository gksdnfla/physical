# 물리공식
------

기체에 관한 공식

- P = 압력
- F = 힘
- A = 면적
- V = 분자
- ∝ = 정비례

```
P = F/A [ N/m² = N/m² = Pa ]
```

> 보일의 법칙

```
P ∝ 1/V
P(처음) * V(처음) = P(나중) * V(나중)
```

> 샤를의 법칙

```
T ∝ V
온도 ∝ 부피

V(나중) = V(처음) + V(처음) * 1 / 273
```

> 보일의 법칙 + 샤를의 법칙

```
P(처음)*V(처음)/T(처음) = P(나중)*V(나중)/T(나중)
```

온도에 관한 공식

- Q = 열량
- T = 절대온도
- t = 섭씨온도
- c = 비열
- m = 질량
- △ = 변화

```
T = t + 237
```

> 비열

```
c = Q / m * △T
```

여러 가지 물질의 비열

|        물질      |   납  |  구리 |  철  | 화강암 |  공기 | 목재 | 얼음 |  물  |
|:---------------:|:----:|:----:|:----:|:----:|:----:|:---:|:---:|:---:|
| 비열(kcal/kg*△T) | 0.03 | 0.09 | 0.11 | 0.19 | 0.25 | 0.4 | 0.5 |  1  |

> 열용량

```
C = c * m [ kcal / kg * △T * kg = kcal/△T ]
```
