# 압력
-------

## 기체에 관한 압력

압력이란 일정한 면적에 작용하는 힘.

- P = 압력
- F = 힘
- A = 면적
- V = 분자
- ∝ = 정비례

공식

> 힘이 커질수록 압력은 커진다.면적이 작을 수록 압력은 커진다.

```
P = F/A [ N/m² = N/m² = Pa ]
압력 = 힘/면적
```

보일의 법칙

> 압력이 커지면 부피가 작아진다.

```
P ∝ 1/V
P(처음) * V(처음) = P(나중) * V(나중)
압력(처음) * 부피(처음) = 압력(나중) * 부피(나중)
```

샤를의 법칙

> 온도가 높아질수록 부피는 커진다.

```
T ∝ V
온도 ∝ 부피

V(나중) = V(처음) + V(처음) * 1 / 273
부피(나중) = 부피(처음) * 부피(처음) * 1 / 273
```
